# IO输入与按键扫描

输入， GPIO其中一个功能。

读取接在IO口上的电平状态。1 或 0。

初始化分析


试验1
先调试一个IO口的输入，如果状态是0，在数码管显示0 ，如果是高电平1， 按键松开，就在数码管显示1.

如果没有数码管，可以控制一个LED的亮灭。
